
+ 图灵完备只需要“顺序+分支+循环+函数+变量+I/O”六件套
+ 工业强度才需要模块、类型、性能、并发、生态

+ 图灵完备最低配清单 ✅
  
    | 机制 | 作用 | Lox 里对应 |
    |---|---|---|
    | 顺序 | 复合计算 | 代码块 { … } |
    | 分支 | 任意判断 | if/else |
    | 循环 | 无限重复 | while / for |
    | 子程序 | 抽象+复用 | fun + return |
    | 存储 | 可变量 | var + 赋值（Lox 在表达式里补了 =）|
    | I/O | 与外界交互 | print / clock |


    | 如果还想…      | 需要加                                 |
    | ---------- | ----------------------------------- |
    | **多文件/模块** | `import`/`export` 或 namespace       |
    | **静态类型**   | 类型标注 + 检查器（可参考 Crafting 第三部分“类型篇章”） |
    | **性能**     | 字节码 + VM（书里第二套实现 clox 就是）           |
    | **并发**     | goroutine、actor、async/await、线程 + 锁  |
    | **元编程**    | 宏、反射、注解、模板                          |
    | **生态**     | 包管理器、标准库、文档工具、LSP                   |
