# Git Stash 速查表

| 场景                      | 命令                                  | 说明                                                    |
| ------------------------- | ------------------------------------- | ------------------------------------------------------- |
| **保存当前改动**          | `git stash push -m "描述"`            | 工作区+暂存区一起打包，工作目录变干净；简写 `git stash` |
| **查看存了哪些包裹**      | `git stash list`                      | 显示 stash 栈，最新为 `stash@{0}`                       |
| **恢复并删除最新包裹**    | `git stash pop`                       | 冲突时手动解决，再 `git add/commit`                     |
| **仅恢复不删除**          | `git stash apply`                     | 等价于 `apply + drop` 的两步拆开                        |
| **恢复指定包裹**          | `git stash apply stash@{1}`           | 编号从 `list` 里看                                      |
| **删除指定包裹**          | `git stash drop stash@{1}`            | 无需恢复直接扔掉                                        |
| **只 stash 部分文件**     | `git stash push -m "备注" -- src/`    | 其余文件保持原样                                        |
| **从 stash 直接开新分支** | `git stash branch 新分支名 stash@{0}` | 恢复成功自动删除该 stash，失败不删                      |
| **清空整个 stash 栈**     | `git stash clear`                     | 一键清空，不可恢复，慎用                                |

> 记忆口诀：**push 存，list 看，pop 取，drop 扔，branch 直接开新坑**
