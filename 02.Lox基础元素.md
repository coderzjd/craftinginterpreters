### 一、数据类型
+ Booleans——没有逻辑就不能编码
    ```js
    true;  // Not false.
    false; // Not *not* false.
    ```
+ Numbers——Lox只有一种数字：双精度浮点数
    ```js
    1234;  // An integer.
    12.34; // A decimal number.
    ```
+ Strings——一个字符串字面量
    ```js
    "I am a string";
    "";    // The empty string.
    "123"; // This is a string, not a number.
    ```
+ Nil——还有最后一个内置数据

### 二、表达式
+ 算术运算
    ```js
    add + me;
    subtract - me;
    multiply * me;
    divide / me;
    ```
+ 比较与相等
    ```j
    less < than;
    lessThan <= orEqual;
    greater > than;
    greaterThan >= orEqual;
    ```
+ 逻辑运算
    - 非
        ```js
        !true;  // false.
        !false; // true.
        ```
    - and表达式
        ```js
        true and false; // false.
        true and true;  // true.
        ```
    - or表达式
        ```js
        false or false; // false.
        true or false;  // true.
        ```
    - 优先级与分组
        ```js
        var average = (min + max) / 2;
        ```
### 三、express 和 statement
+ express 必须有一个值
+ statement 触发一个effect
+ { }: 表达代码块
+ expression后跟分号（;）可以将expression提升为statement状态
  
### 四、变量声明
+ var语句声明变量。如果你省略了初始化操作，变量的值默认为nil
    ```js
    var imAVariable = "here is my value";
    var iAmNil;
    ```
### 五、控制流
+ if语句根据某些条件执行两条语句中的一条
    ```js
    if (condition) {
        print "yes";
    } else {
        print "no";
    }
    ```
+ while循环
    ```js
    var a = 1;
    while (a < 10) {
        print a;
        a = a + 1;
    }
    ```
+ for循环
    ```js
    for (var a = 1; a < 10; a = a + 1) {
    print a;
    }
    ```
### 六、函数
+ 定义运行函数
    ```js
    fun returnSum(a, b) {
    return a + b;
    }
    returnSum(1,2)
    ```
+ 执行到达代码块的末尾而没有return语句，则会隐式返回nil
+ 支持闭包

### 七、lox中的类
+ 类定义、定义类实例、init初始化、
    ```js
    class Breakfast {
    init(meat, bread) {
        this.meat = meat;
        this.bread = bread;
    }

    // ...
    }
    var baconAndToast = Breakfast("bacon", "toast");
    baconAndToast.serve("Dear Reader");
    ```
+ 类的继承、调用父类init函数
    ```js
    class Brunch < Breakfast {
    init(meat, bread, drink) {
        super.init(meat, bread);
        this.drink = drink;
    }
    }
    ```
