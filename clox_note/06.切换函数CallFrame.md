### 1、函数实现
+ 通过切换CallFrame实现函数调用
```bash
    CallFrame *frame = &vm.frames[vm.frameCount++];
    frame->function = function;
    frame->ip = function->chunk.code;
    frame->slots = vm.stackTop - argCount - 1;
```
### 2、函数实现局部变量
+ 语义分析阶段： 计算局部变量的对应的vm stack变量地址 index
+ 运行阶段： 通过 index 直接从 stack总取值
+ 编译器把每个局部变量提前拍平成“栈槽号”，运行时解释器就当数组用，帧切换只需改基址指针，于是函数调用和局部变量都变成 O(1) 的数组操作
